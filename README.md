NishaTech CyberOps 🛡️
Autonomous Cyber Threat Routing & Mitigation System

NishaTech CyberOps is an AI-powered autonomous incident routing system. It utilizes decentralized multi-agent intelligence—featuring Triage, Mail Guard, and Net Defender nodes—to instantly detect, analyze, and mitigate cyber threats like financial fraud, ensuring robust, real-time protection for organizational infrastructure.

🚀 Key Features
Multi-Agent Coordination: Real-time routing via specialized nodes (Triage, Mail Guard, Net Defender).

Autonomous Sandbox Scan: Automated cross-agent sweeps to isolate and analyze infrastructure anomalies.

Financial Fraud Mitigation: Instant detection and response to spoofed banking and phishing attempts.

Co-Pilot Integration: Interactive AI assistant for security operations and threat management.

Professional UI: Minimalist dashboard designed for high-stakes cybersecurity ops.

🛠️ Tech Stack
Backend: Flask (Python)

Frontend: HTML5, CSS3, JavaScript

Integration: Aicoo API / Infrastructure

Design: Minimalist Corporate UI

📋 Installation & Setup
Clone the repository:

Bash
git clone https://github.com/YOUR_USERNAME/NishaTech-CyberOps.git
cd NishaTech-CyberOps
Install dependencies:

Bash
pip install -r requirements.txt
Run the application:

Bash
python app.py
Access the platform:
Open your browser and navigate to [http://127.0.0.1:5000](http://127.0.0.1:5000)

💡 Role of Aicoo
Aicoo acts as the coordination layer of NishaTech. It enables the agents (Triage, Mail Guard, Net Defender) to maintain distinct identities, communicate threat context, and route incident requests autonomously to the appropriate security handlers.
